WebX+ Domains is the official registrar for the WebX+ DNS.


# How to use WebX+

In order to use WebX+, you need a browser that supports custom DNS, we recommend [Bussinga](https://github.com/codingMASTER398/bussinga).

If you are using Bussinga, then the DNS servers are already configured

For other browsers, you need to find the option to set a custom DNS and add the following:

- **Primary DNS**: `webxdns.votemanager.xyz`
- **Secondary DNS**: `webxdns.blocklycode.org`

Most browsers will only allow you to enter 1 DNS server, in that case just add the primary one. If you are experiencing issues, try adding the secondary DNS instead.

All normaly WebX domains are available via the WebX+ DNS
